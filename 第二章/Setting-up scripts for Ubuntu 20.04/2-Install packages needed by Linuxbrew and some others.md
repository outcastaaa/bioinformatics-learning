-L,--location:如果服务器报告请求的页面已移动到其他位置（用location:header和3xx 响应代码），此选项将使curl在新位置上重新执行请求。

最简单也可能是最常见的例子，就是针对某个软件的某个漏洞的补丁。某个软件的新版本发现了漏洞，通过修补源代码后可以修复；\
但此软件的旧版本因为源代码不同，而不能通过同样的修补来修复，这时就需要针对旧版本的软件来进行源代码修补了。
